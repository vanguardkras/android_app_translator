Порядок действий.

1. Базовая настройка приложения.

Для начала нужно сгенерировать ключ для переводчика на яндексе:
https://translate.yandex.ru/developers/keys

Данный ключ нужно вставить в значение поля key файла конфигурации settings.ini

Можно какое-то время пользоваться и моим ключом.

2. Дополнительная настройка приложения

В файле конфигурации settings.ini измените initial_lang для того, чтоб изменить,
с какого языка будет осуществляться перевод.

В этом же файле в разделе [language_list] перечислите список языков, на которые нужно
осуществить перевод, используя их сокращенные двухбуквенные наименования в нижнем
регистре. Каждый язык должен содержать числовой или символьный уникальный индекс.
Можно просто перечислить их по порядку, как в примере.

3. Запуск приложения.

Приложение должно быть запущено с ПК с прямым интернет соединением.

положите в папку C:\translate папку values вашего приложения только с теми файлами,
которые нужно перевести.

Запустите файл run.bat и ждите окончания процесса перевода.
Может занять несколько минут.
